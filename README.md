## ✨ Features

- 📅 Format dates in any locale
- ⏰ 12h or 24h time support
- 🔧 Fully customizable format string (`dd/mm/yyyy`, etc.)
- 📦 Structured return format (`{ day, month, year, ... }`)
- 🌐 Lightweight, tree-shakable & TypeScript ready

---

## 📦 Installation

```bash
npm install date-react
```
